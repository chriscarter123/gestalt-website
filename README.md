# Gestalt — Marketing Website

Premium marketing landing page for the [Gestalt](https://github.com/chriscarter123/gestalt) AR art discovery platform.

Built with vanilla HTML, CSS, and JavaScript following [taste-skill](https://github.com/Leonxlnx/taste-skill) design directives.

## Run Locally

```bash
# Any static file server works
python3 -m http.server 8765
# then open http://localhost:8765
```

## Design

- **Vibe:** Soft Structuralism — silver-grey backgrounds, massive Outfit typography, diffused ambient shadows
- **Layout:** Asymmetrical Bento grid, editorial split sections
- **Motion:** Scroll-triggered fade-up reveals via IntersectionObserver, kinetic marquee
- **Cards:** Double-Bezel nested architecture (outer shell + inner core)
- **Accent:** Gestalt brand green `#3ECF6A`

## Sections

1. Floating glass pill navigation
2. Asymmetric hero with AI-powered art discovery imagery
3. Kinetic marquee text band
4. Features bento grid (AI Recognition, GPS, QR, Audio, Multi-Venue)
5. How It Works (Point, Discover, Listen)
6. Technology stack (editorial split)
7. CTA / Download
8. Footer
